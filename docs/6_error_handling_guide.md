# 버스타볼까 에러 가이드 (ERROR_GUIDE)

## 1. 글로벌 에러 처리
| 상황 | 사용자 메시지 | 기술적 처리 |
| :--- | :--- | :--- |
| API 통신 실패 | "버스 정보를 불러오지 못했어요. 잠시 후 다시 시도해주세요." | Toast 알림 + 재시도 버튼 |
| 위치 권한 거부 | "내 위치를 확인하려면 위치 권한이 필요합니다." | 위치 권한 요청 모달 |
| 검색 결과 없음 | "해당 경로를 찾을 수 없습니다." | Empty State 컴포넌트 표시 |

## 2. 엣지 케이스 (Edge Cases)
- **막차 시간 지난 후 검색**: "운행 시간이 종료되었습니다" 명시적 표시
- **네트워크 오프라인**: "인터넷 연결이 불안정합니다" (서비스 워커 캐싱 고려 가능)
- **환승 정보 없음**: 직통 노선이 없을 경우, "근처 다른 정류장" 추천 텍스트 표시

## 3. UI 로딩 상태 (Skeleton)
- **지도 로딩 중**: 로딩 스피너 + "지도를 불러오는 중입니다"
- **경로 리스트**: 스켈레톤 UI (깜빡임 최소화)
